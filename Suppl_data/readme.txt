The zip file contains the "Suppl_data" of the PNTD-D-25-01192 PLOSNTD submitted manuscript for publication.

There are 7 subfolders: Aedes, Anopheles, Glossina, Sandflies, Stomoxys, Tabanus and Triatoma.

1. subfolder Aedes.

Male and female Aedes 16 wing landmarks raw coordinates allowing the morphometric comparisons of Ae. albopictus with Ae. scutellaris, Ae. aegypti with Ae. scutellaris, and Ae. albopictus with Ae. aegypti.

The name of each file contains the following abbreviations
LM landmarks (their raw coordinates)
F females
M males
aeg = Ae. aegypti
alb = Ae. albopictus
scu = Ae. scutellaris
Each file name ends by the sample sizes for each species. For instance "LM_Malb-scu_41-36" means Landmarks of 41 male Aedes albopictus followed by the landmarks of 36 male Aedes scutellaris. 

2. subfolder Anopheles. There are two subsubfolders "min-har" and "saw-pse".
min = An. minimus
har = An. harrisoni
saw = An. sawadwongporni
pse = An. pseudowillmori

2.1 subsubfolder "min-har"
16 wing landmarks of female An. minimus Theobald, 1901 and An. harrisoni Harbach & Manguin, 2007. They are based on the same number of landmarks (16) as for the Aedes species, and collected in the same order.

Each file contains the raw coordinates of two species, ordered according to the name of the file. For instance:
"Female_67 min 22 har.txt" contains the raw coordinates of 16 landmarks for 22 An. minimus followed by the raw coordinate of the same landmarks (in the same order) for 22 harrisoni

2.2 subsubfolder "saw-pse"
The raw coordinates correspond to 17 landmarks. Here, different operators, T and JP, used a different sampling of the mosquitoes: 41,36 and 26,28, respectively. For instance:
The "Female_sawad_pseudo_42-36.txt" file contains the landmarks raw coordinates of 42 An. sawadwongporni followed by 36 raw coordinates of 36 An. pseudowillmori. These coordinates were annotated by user "T".

3. subfolder Glossina.

Two subspecies of Glossina palpalis from Ivory Coast (Côte d’Ivoire): G. palpalis gambiensis (Gpg) and G. palpalis palpalis (Gpp) [9]. Eleven landmarks were annotated by two operators: B(erte) and TA(Ta).

For instance:
The "Gpg_40-40_Berte-TA.txt" file contains the raw coordinates of 40 G. p. gambiensis annotated by the user "Berte" followed by the raw coordinates of exactly the same 40 G. p. gambiensis but annotated  by the user "Ta".

The "Gpg-Gpp_40-18_Berte_F.txt" file contains 40 (female) G. p. gambiensis followed by the raw coordinates of 18 G. p. palpalis, annotated by the operator "Berte".
The "HuitLM_Gpg-Gpp_40-18_Berte_F.txt" file contains the raw coordinates of the same specimens as the "Gpg-Gpp_40-18_Berte_F.txt" file, but for only 8 landmarks instead of 11 (LM 1, LM2 and LM11 where removed).

4. subfolder Sandflies (Psychodidae)

Shod = Sergentomyia hodgsoni (Sinton, 1933) 
Pstan = Phlebotomus stantoni (Newstead 1914)

One file only, , containing the raw coordinates of 12 landmarks of 44 P. stantoni followed by the raw coordinates of 35 S. hodgsoni.

5. subfolder Stomoxys.

One file only, containing the raw coordinates of 10 landmarks for male 35 Stomoxys pullus (SPM) followed by 34 male Stomoxys uruma (SUN).

6. subfolder Tabanus.

Tabanus megalops ("me") and Tabanus striatus ("st"). 

One file only, "mest129-85.txt", containing the raw coordinates of 22 landmarks of 129 T. megalops followed by the raw coordinates of 85 T. striatus. 

7. subfolder Triatoma.

The North American Triatoma protracta (Uhler) (1894).: T. p. woodi and T. p. protracta. 13 landmarks.

One file only, "20-24-29-landmarks.txt", containing the raw coordinates of 13 landmarks of 20+24=44 T. p. protracta followed by the raw coordinates of 29  T. p. woodi.
